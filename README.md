# ShiftAi Systems Ltd — Corporate Website
**Live Domain:** [https://shiftaiconsulting.co.uk](https://shiftaiconsulting.co.uk)

## 🏢 About
This repository houses the static frontend for **ShiftAi Systems Ltd**, an AI architecture consultancy based in London, UK. 

Built using lightweight HTML5 and Tailwind CSS, this site serves as the portfolio storefront for our enterprise-grade Agentic AI solutions, emphasizing fast load times, zero-dependency deployment via GitHub Pages, and a secure, tracker-free footprint.

---

## 🏗️ Our Reference Architectures
We do not build chatbot wrappers. We embed within risk and engineering teams to map data lineage, deploy Human-in-the-Loop (HITL) execution breakpoints, and automate CI/CD pipeline evaluations. 

Review our open-source, DevSecOps-hardened banking architectures:

1. 🏦 **[Commercial Credit Underwriting](https://github.com/TechIfat/credit-underwriting-ai):** Bypassing OCR via Claude Vision & isolating LLM arithmetic via MCP.
2. ⚙️ **[Enterprise Agentic Platform](https://github.com/TechIfat/my-enterprise-platform):** LangGraph DAGs with strict Human-in-the-Loop (HITL) transaction breakpoints.
3. ⚖️ **[Regulatory Gap Analyzer](https://github.com/TechIfat/regulatory-gap-analyzer):** Multi-Document Reasoning with strict XML context isolation and forced Pydantic citations.
4. 💸 **[AML False-Positive Triage](https://github.com/TechIfat/aml-batch-processor):** Asynchronous FinOps using Anthropic's Message Batch API.
